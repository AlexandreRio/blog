Comme je suis en vacance j'ai eu le temps de revoir mon script de blagues de geek! Le  code est donc bien plus propre et léger, de plus l'affichage a même été refait! Voilà le code source: [bash]#!/bin/bash #                           Script d'affichage de blague de geek V2.0 #                                   Necessite le paquet zenity ################################################################################################# #                           Par Orbital: http://orbital-project.kegtux.org/blog/ #                               http://twitter.com/Rio_Alexandre ################################################################################################# let "continuer=1" while [ $continuer -eq 1 ] do message=`wget -O  - http://orbital-project.kegtux.org/programmes/bdg_script.php` zenity --info --text="$message" --title="Blague de geek" zenity --question --ok-label="une autre!" --cancel-label="Quitter" --text="Continuer ?" if [ $? -eq 0 ] then let "continuer = 1" else let "continuer = 0" fi done[/bash] Et le nouveau design: L'encodage merdouille encore un petit peu, quand c'est le cas ou rien de n'affiche et le script vous demande si vous voulez continuer, ou on vous dit que "Toutes les mises à jour sont complètes."
